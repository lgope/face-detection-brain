# face-detection-brain (Back End)

## Technologies

1. nodejs
2. expressjs
3. postgreSQL (for store user data)

## To run this project :

1. Clone this repo
2. Change your directory : `cd face-detection-brain/Back-End-Server`
3. Run `npm install`
4. Run `npm start`
